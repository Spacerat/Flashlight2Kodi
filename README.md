# Kodi/XBMC Control from Spotlight

Launch youtube/soundcloud videos on XBMC through Spotlight, and perform
basic operations e.g. play/pause.

Examples of usage:

	kodiyt Youtube Search
	kodisc Scoundcloud search
	kodi play
	kodi stop
